## \[LeetCode 119] Pascal’s Triangle II

## Problem Statement

Given an integer `rowIndex`, return the **rowIndex-th row** of Pascal’s Triangle.

In Pascal’s Triangle:

- The top element is `1` (row 0).
- Each element is the sum of the two elements directly above it.
- The row at index `n` contains `n + 1` elements.

---

## Examples

### Example 1:

**Input:**

```
rowIndex = 3
```

**Output:**

```
[1, 3, 3, 1]
```

---

### Example 2:

**Input:**

```
rowIndex = 0
```

**Output:**

```
[1]
```

---

### Example 3:

**Input:**

```
rowIndex = 1
```

**Output:**

```
[1, 1]
```

---

## Constraints

- `0 <= rowIndex <= 33`
