# ✅ Count the Number of Set Bits

### 🧾 Problem Statement

Given a non-negative integer `n`, return the **number of 1s (set bits)** in its binary representation.

---

### 🔍 Examples

**Input:**

```
n = 5
```

**Explanation:**
Binary = `101` → 2 set bits
**Output:**

```
2
```

**Input:**

```
n = 0
```

**Output:**

```
0
```

**Input:**

```
n = 15
```

**Explanation:**
Binary = `1111` → 4 set bits
**Output:**

```
4
```

---

### 🔒 Constraints:

* `0 <= n <= 2^31 - 1`
