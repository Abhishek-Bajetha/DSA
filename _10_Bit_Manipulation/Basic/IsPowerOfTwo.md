# ✅ Check if a Number is a Power of Two

### 🧾 Problem Statement

Write a function that returns `true` if the given number `n` is a power of 2. Otherwise, return `false`.

---

### 🔍 Examples

**Input:**

```
n = 16
```

**Output:**

```
true
```

**Input:**

```
n = 18
```

**Output:**

```
false
```

**Input:**

```
n = 1
```

**Output:**

```
true
```

---

### 🔒 Constraints:

* `0 <= n <= 2^31 - 1`
