### ✅ Check if a Number is Odd

## 🧾 Problem Statement

Write a function to **check whether a given number `n` is odd or even** using bit manipulation.

---

## 🔍 Example 1:

**Input:**

```
n = 5
```

**Output:**

```
Odd
```

---

## 🔍 Example 2:

**Input:**

```
n = 8
```

**Output:**

```
Even
```

---

## 🔒 Constraints:

* `0 <= n <= 10^9`
